Abstract

The concept of the project is to design a measurments station called "Archimedes Bathtub" in which an object with uknown mass, density or volume can be recognized.

3D Design (Using Fusion 360)

Modelling the "Bathtub" basic shape by first sketching the profile, then extruding:

Then closing the bottom of the bathtub by extruding the inner base rectangular profile

Then refinning the shape by using the fillet tool on the four side edges, after that, creating a decoration for the rim by first sketching the profile then extruding

Sketching the profile for the columns bases, and mirroring them to the four sides

Then modelling the columns details by first sketching the profile, then applying the rectangular pattern tool, and eventually extruding it:

Refinning the the columns tops and bottoms using the fillet tool

Then by using the text tool we can add the word "EUREKA!" after creating a sketch on one of the bathtub sides, then applying the option "Text Explode" to make the text engraved on the surface
